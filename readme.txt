[1] INSTALLATION
    [#] Start mysql and apache server in xampp.
    [#] Navigate to PHPMYADMIN.
    [#] Create new database "db_conspring".
    [#] Import database file "db_conspring.sql".
    [#] Navigate to "c:\xampp\htdocs" directory and create new folder "ConSpring".
    [#] Extract the project "ConSpring.zip" to "c:\xampp\htdocs\ConSpring" directory.
    [#] Navigate to "localhost/ConSpring" on the browser to run the project.
