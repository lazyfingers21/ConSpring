INSTALLATION
    [1] Start mysql and apache server in xampp.
    [2] Navigate to PHPMYADMIN.
    [3] Create new database "db_conspring".
    [4] Import database file "db_conspring.sql".
    [5] Navigate to "c:\xampp\htdocs" directory and create new folder "ConSpring".
    [6] Extract the project "ConSpring.zip" to "c:\xampp\htdocs\ConSpring" directory.
    [7] Navigate to "localhost/ConSpring" on the browser to run the project.
